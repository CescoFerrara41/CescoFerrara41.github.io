2D Video Game Development

A custom-built 2D game featuring enemy AI behavior, projectile systems, and polished gameplay mechanics developed from the ground up.

Overview

This project is a 2D game developed entirely from scratch with a focus on creating responsive enemy behavior, engaging combat mechanics, and a scalable architecture for adding new content. The game demonstrates advanced AI patterns, complex collision systems, and smooth player controls.

Gameplay Systems

  • Enemy AI with dynamic movement patterns and adaptive attack phases
  • Robust projectile and collision detection systems
  • Responsive player movement and combat mechanics
  • Scene-based progression with state management
  • Health and damage calculation systems
  • Visual feedback and particle effects

Tech Stack

Unity Engine

2D game framework with physics and rendering

C# Scripting

Core gameplay logic and systems

Custom AI

Behavior trees and state machines

Sprite Animation

Frame-based character animations

Technical Highlights

  • Enemies dynamically adjust behavior based on player position and health
  • Shotgun-style projectile patterns with spread calculations
  • Modular, expandable architecture for new enemy types and abilities
  • Efficient object pooling for projectiles and effects
  • Persistent state management across scenes
  • Performance optimization for smooth 60 FPS gameplay

Challenges & Solutions

One of the main challenges was creating AI that felt challenging but fair. I solved this by implementing multiple difficulty phases that gradually introduce new attack patterns, giving players time to learn and adapt. The projectile system required careful optimization to handle dozens of simultaneous projectiles without performance drops, which I achieved through object pooling and efficient collision detection.
